# Brute Force

## 개요
Brute Force 공격은 가능한 모든 경우의 수를 하나씩 대입하여 정답을 찾는 공격 방식입니다.

---

## 핵심 개념
- 특정 값을 모를 때 모든 경우를 시도하여 결과를 얻는 방식입니다.
- 주로 로그인 비밀번호, 암호 키, 인증 정보 등을 대상으로 수행됩니다.

---

## 공격 방식 설명
공격자는 자동화 도구를 이용하여 가능한 입력값을 순차적으로 시도합니다.
정답이 나올 때까지 반복적으로 요청을 보내는 방식으로 동작합니다.

---

## 예시
- 비밀번호가 4자리 숫자일 경우  
  → 0000부터 9999까지 모두 시도합니다.

- 웹 로그인 공격 예시  
  → admin / 1234  
  → admin / 12345  
  → admin / password  
  → admin / qwerty  
  → ... 반복 시도

---

## 특징
- 성공 확률이 높지만 시간이 오래 걸립니다.
- 경우의 수가 많을수록 공격 시간이 기하급수적으로 증가합니다.

---

## 대응 방안 (간단)
- 로그인 시도 횟수 제한
- CAPTCHA 적용
- 계정 잠금 정책 적용

---

## 정리
- Brute Force 공격은 가능한 모든 경우를 시도하여 정답을 찾는 무차별 대입 공격 방식입니다.
